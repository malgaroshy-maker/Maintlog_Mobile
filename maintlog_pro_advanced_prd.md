
# MaintLog Pro – Advanced Product Requirements Document (PRD)

Version: 2.0
Author: MaintLog Concept by User
Purpose: Production‑grade specification for building a Maintenance Daily Activity Logbook Application.

---

# 1. Product Overview

MaintLog Pro is a **maintenance activity reporting system** designed to replace traditional paper logbooks used in factories.

The application allows maintenance engineers to log daily work activities in a **spreadsheet-style interface**, track spare parts usage, monitor machine downtime, and generate reports.

The system is optimized for **industrial environments**, where speed, clarity, and reliability are critical.

Primary platform:
- Mobile (Android first)
- Optional Desktop (Windows)
- Cloud sync via Supabase

---

# 2. Core Product Goals

1. Replace paper maintenance logbooks.
2. Allow engineers to log interventions quickly.
3. Provide historical analytics for maintenance managers.
4. Track spare parts usage.
5. Enable AI assisted log entry and data analysis.
6. Provide clean printable reports for management.

# 5. Sections Architecture

Factories are divided into **Sections**.

Examples:

- Filling
- Packaging
- Utilities
- Warehouse

Each section has its own:

- Machines
- Engineers
- Spare parts
- Reports

This ensures **data isolation** between departments.

---

# 6. Shift Structure

Each day contains three shifts:

1. Night Shift
2. Morning Shift
3. Evening Shift

Each shift contains:

- Assigned engineers
- Multiple maintenance entries

Important rule:

Engineers are selected **once per shift**, not per entry.

---

# 7. Logbook Interface (Most Important Feature)

The core UI is a **spreadsheet style logbook**.

Rows represent maintenance activities.

Columns:

Machine | Line | Work Description | Total Time | Spare Parts | Quantity | Notes

Example row:

CFA | 1 | Replaced seal kit due to leakage | 90m | Seal Kit CFA | 1 | Leak fixed

---

# 8. Spreadsheet Behavior

Essential behaviors:

### Cell Editing

Tap a cell to edit.

Different columns have different input types.

Machine → dropdown  
Line → multi select checkbox  
Work Description → rich text editor  
Total Time → duration input  
Spare Parts → part selector popup  
Notes → free text

Design a spreadsheet style maintenance logbook interface for a factory maintenance reporting app. The screen should resemble an industrial daily maintenance report sheet with rows and columns.
